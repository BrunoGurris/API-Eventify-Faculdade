## Instalação do Projeto
```
1- Verifique se vc possui o Composer instalado no computador: https://getcomposer.org/
2- Clone o repositorio para seu computador: https://github.com/BrunoGurris/API-Eventify-Faculdade.git
3- Abra o projeto utilizando uma IDE, por exemplo: VS CODE
4- Na pasta do projeto, execute o comando: composer install, para que baixe todas as dependencias
5- Após o comando ter finalizado, execute o comando: php artisan serve, para rodar o projeto
6- No seu navegador, entre no link: http://localhost:8000/
7 OBS: Existe um arquivo .env, onde há todas as configurações. É necessario possuir um DB e fazer a configuração do mesmo
```
